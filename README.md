# Personal Blog Engine

A lightweight, client-side blog engine with Markdown support and admin panel.

## Features
- Create, edit, delete posts via admin panel
- Markdown rendering with syntax highlighting (highlight.js)
- Tagging system
- Responsive design (Tailwind CSS)
- Client-side routing (hash-based)
- LocalStorage persistence
- SEO-friendly URLs via slug
- No backend required

## Tech
- HTML5, Tailwind CSS, Vanilla JS
- Marked.js for Markdown
- Highlight.js for code blocks
- LocalStorage for data persistence

## Usage
1. Open `index.html` in browser
2. Click "Admin" to create posts
3. Publish and view on homepage
4. Share post URLs (hash-based)

## Deploy
Static site - push to GitHub Pages.
